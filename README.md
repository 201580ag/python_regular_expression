# python_regular_expression

정규 표현식 예제
```\d: 숫자에 일치합니다. [0-9]와 동일합니다.
\D: 숫자가 아닌 문자에 일치합니다. [^0-9]와 동일합니다.
\w: 숫자와 문자에 일치합니다. [a-zA-Z0-9_]와 동일합니다.
\W: 숫자와 문자가 아닌 문자에 일치합니다. [^a-zA-Z0-9_]와 동일합니다.
\s: 공백 문자에 일치합니다. 공백, 탭, 줄 바꿈 등을 포함합니다.
\S: 공백 문자가 아닌 문자에 일치합니다.
\b: 단어 경계에 일치합니다. 단어 경계는 단어와 비단어 문자 사이에 있는 위치를 나타냅니다.
.: 줄 바꿈 문자를 제외한 모든 문자에 일치합니다.
*: 0회 이상 반복을 나타내는 메타 문자입니다.
+: 1회 이상 반복을 나타내는 메타 문자입니다.
?: 0 또는 1회의 발생을 나타내는 메타 문자입니다.
[]: 괄호 안에 포함된 문자 중 하나에 일치합니다.
[abc]: 'a', 'b', 'c' 중 하나에 일치합니다.
[0-9]: 0부터 9까지의 숫자 중 하나에 일치합니다.
[^]: 괄호 안에 포함되지 않은 문자에 일치합니다.
[^abc]: 'a', 'b', 'c'가 아닌 문자에 일치합니다.
|: 또는을 의미하는 메타 문자입니다.
apple|banana: 'apple' 또는 'banana'에 일치합니다.
^: 문자열의 시작을 나타내는 메타 문자입니다.
^Hello: 'Hello'로 시작하는 문자열에 일치합니다.
$: 문자열의 끝을 나타내는 메타 문자입니다.
World$: 'World'로 끝나는 문자열에 일치합니다.```
